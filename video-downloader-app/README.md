# Project 5: Video Downloader App (Kivy GUI)

A desktop GUI application built with Kivy that downloads videos using yt-dlp.

## Setup

1. Create a virtual environment and activate it:
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Run

```bash
python main.py
```

A Kivy window will open where you can paste a video URL and click Download.
