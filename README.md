# API Integration Course – Modern Python with FastAPI

This repository contains 5 projects from the [API Integration Course](https://youtu.be/0sOvCWFmrtA).

## Projects

| # | Folder | Description | Stack |
|---|--------|-------------|-------|
| 1 | `translation-service/` | Real-Time Translation Service | FastAPI + OpenAI GPT-4 |
| 2 | `content-generator-service/` | AI Content Generator & Sentiment Analyzer | FastAPI + OpenAI GPT-4 |
| 3 | `lyrics-extraction-service/` | Lyrics Extraction Web App | FastAPI + lyrics.ovh API |
| 4 | `image-to-text-service/` | Image to Text OCR | FastAPI + PyTesseract |
| 5 | `video-downloader-app/` | Video Downloader Desktop App | Kivy + yt-dlp |

## Getting Started

Each project has its own `README.md` with setup and run instructions. Navigate into the desired project folder to begin.
